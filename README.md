# Agent OS - 智能体操作系统官方站

agentos.md 官方网站，智能体操作系统行业权威入口。

## 功能模块

- **首页**：Agent OS 行业全景图，展示主流Agent OS项目
- **资源导航**：教程文档、开源项目、社区资源、开发工具汇总
- **博客**：行业动态、技术分析、项目评测文章

## 技术栈

- React 18 + Vite
- React Router 6
- 响应式设计，支持移动端

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 部署

推荐部署到 Vercel 或 Netlify，支持自动部署和SSL证书。
